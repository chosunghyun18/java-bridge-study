# 미션 - 다리 건너기

1차 제출  : 13:30 ~ 17:20 : 3 시간 50 분 소요

시스템 외부 값 입력을 예상 view 우선 처리 -> controller 도입으로 요구 사항 충족
makeBridge 사용법 숙지 , 디버깅 + 테스트 코드 분석  , 예시로 준 테스트 코드 프로 덕션 코드에 바로 적용 후 리펙 : ~ 15 : 30
문서 지속적으로 업데이트 + 나머지 클라스 도입 완료 ~ 17:20
